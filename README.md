
# ✅ To-Do List

Lista de tarefas desenvolvida com HTML, CSS e JavaScript para treinar meus conhecimentos em desenvolvimento web.

Este projeto é um exercício prático que me permitiu aplicar conceitos básicos como manipulação do DOM, eventos, armazenamento local (localStorage) e boas práticas de separação de arquivos.

---

## ✨ Funcionalidades

- ✅ Adicionar novas tarefas
- ✅ Marcar tarefas como concluídas
- ✅ Remover tarefas da lista
- ✅ Armazenar as tarefas no navegador usando `localStorage`

---

## 🚀 Tecnologias Utilizadas

- HTML5
- CSS3
- JavaScript (puro)

---

## 🛠️ Como Executar o Projeto Localmente

1. **Clone o repositório:**

```bash
git clone https://github.com/ctorres4564/to-do-list.git
````

2. **Acesse a pasta do projeto:**

```bash
cd to-do-list
```

3. **Abra o arquivo `index.html` em seu navegador:**

Você pode clicar duas vezes sobre o arquivo ou usar uma extensão de servidor local como o Live Server (VS Code).
